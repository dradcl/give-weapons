# give-weapons
A Sven-Coop server plugin for getting weapons

**Usage:**

- In-game text chat, type .givewep {weapon_weaponname}
- Console, type as_command .givewep {weapon_weaponname}

List of weapons that can be used:

- weapon_9mmhandgun
